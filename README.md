## 🎮 Como usar
```
1. Clonando o repositório 
$ git clone https://github.com/clestonv/robot_api_servrest.git

2. Acessando o diretório do projeto 
$ cd robot_api_servrest

3. Executando os testes
$ robot -d /result/ tests/

```
